=== COMPTE RENDU DE PROJET ===
Ce répertoire contient 2 sous-répertoires
	- "documentation technique" contient le code source de notre compilateur
		Les classes utiles à la compilation sont définies dans le package "generator"
		La grammaire est définie dans le fichier "Projet.xtext"
		La classe "Launcher.java" permet de lancer le compilateur directement plutôt que via un serveur
		Le package "server" contient la classe "SimpleHttpServer.java" qui défini notre serveur java pour l'interface web
	- "interface" contient de quoi tester le compilateur. Un fichier README.txt y explique plus en détail la méthode à suivre.

Il contient en plus de cela le compte rendu de notre projet au format PDF.
